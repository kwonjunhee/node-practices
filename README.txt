Node Practices

1. 패키지
    - 완전한 어플리케이션(nodedaemon, babel, webpack)
    - 프로젝트에서 사용하는 모듈(라이브러리)

2. 의존성 (Dependency)
    - 개발하느 프로젝트(어플리케이션, 패키지)에서 설치하고 사용하는 패키지
    - 일반 의존성
      개발하고 있는 프로젝트에서 사용하는 패키지로 꼭 배포에 포함
      $[Project-ex01]
    - 개발 의존성
      개발에 필요하거나 도움이 되는 패키지(dev tool) 배포에 포함되지 않는다.
